To run this app locally on your system,
1. Install NodeJS and MongoDB on your system.
2. Open command line in the root directory of the project and run: npm i express mongodb mongoose
3. After installing these packages, open MongoDBCompass app.
4. Create a connection on URI: mongodb://0.0.0.0:27017
5. Create a database called 'mydatabase' and a collection called 'subscribe'.
6. Go to porject folder and run command 'node index.js'
7. Open browser and go to 'localhost:8080'
8. The website will open.
9. Scroll down to the bottom, enter email address and click Submit. The email address will be added to the database.
